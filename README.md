# ProductivityPush

A productivity app designed to boost daily motivation and block time-wasting distractions.

## Goals

1. **Daily Motivation & Focus**: Provide daily motivation and focus to get something done every day, with increasing activity levels
2. **Distraction Blocking**: Keep you away from time-wasting apps (YouTube, 9GAG, Facebook, etc.)

## Features (Coming Soon)

- Daily task tracking and motivation
- Progressive activity goals
- Website/app blocking functionality
- Focus time tracking
