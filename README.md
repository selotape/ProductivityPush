# ProductivityPush

A multi-platform productivity solution designed to boost daily motivation and block time-wasting distractions across your devices.

## Goals

1. **Daily Motivation & Focus**: Provide daily motivation and focus to get something done every day, with increasing activity levels
2. **Distraction Blocking**: Keep you away from time-wasting apps (YouTube, 9GAG, Facebook, etc.)

## Getting Started

### Chrome Extension
1. Navigate to `chrome-extension/`
2. Load the extension in Chrome Developer mode
3. See the extension README for detailed setup

### Android App
1. Open `android-app/` in Android Studio
2. Build and install on your Android device
3. Grant required permissions for app monitoring
4. See the app README for detailed setup
